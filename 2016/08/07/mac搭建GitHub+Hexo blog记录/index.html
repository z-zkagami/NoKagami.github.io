好了现在账号有了，叼炸天的名字也有了。咱可以去GitHub上create New repository，Repository name的格式必须是username.github.io,username就是前面强调的你那叼炸天的用户名，如果不是username，会报404错，其它啥的不用填不用选也没关系。创建好了GitHub会生成一个.git的链接，复制它，马上要用到
